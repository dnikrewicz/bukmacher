## General info
Application for betting on sports matches. 
The user has the option of registering, logging in and betting on matches using the available points.
	
## Technologies
Project is created with:
* Java: 17
* Spring Boot: 2.75
* Maven version: 4.0.0
* H2 database
* jquery version 3.4.1
* spring-security
